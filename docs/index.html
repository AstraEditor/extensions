AstraEditor Extension Gallery

This is the official extension gallery, where you can get extensions that aren't limited by the extension sandbox. All of the extensions here are reviewed for safety.

You can also find these extensions in AstraEditor's "Add Extension" menu.

These extensions are not compatible with Scratch.
Projects that use these extensions can't be opened in Scratch or uploaded to the Scratch website. They can still be used in the TurboWarp packager.
Some extensions may not work in AstraEditor Desktop.
To make sure extensions work offline, each release of AstraEditor Desktop comes with a copy of these extensions as of its release date, so some extensions may be outdated or missing. Update to the latest version for the best results.
